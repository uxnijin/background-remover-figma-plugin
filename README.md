# Instant Background Remover

### Erase backgrounds from any image for free, directly inside Figma.

Remove the background from any photo in just one click. Choose an image on your Figma canvas or upload a file from your computer. Our plugin processes everything safely on your device—no servers, no sign-ups, and completely free.

---

## How It Works (3 Simple Steps)
1. **Choose your image**: Select a layer on your Figma canvas, or upload an image file from your computer.
2. **Select Quality**: Choose between **High Quality** (best detail) or **Standard Quality** (faster processing).
3. **Remove Background**: Click the button, let the AI work its magic, and then **Paste it back into Figma** or **Download as a PNG** file!

## Key Features
*   **100% Free**: No API keys, credits, or subscriptions needed.
*   **Private & Secure**: Everything is processed locally in your browser. Your images are never sent to external servers.
*   **Figma-Native Design**: Fits beautifully into your Figma workspace with automatic dark/light mode adjustment.
*   **Dual Ingestion**: Easily switch between canvas selection and local file upload.

---

## Developer Quickstart

This plugin is built using **TypeScript** and **HTML/CSS/JS**.

### 1. Installation
Install project dependencies:
```bash
npm install
```

### 2. Build & Compile TypeScript
Compile TypeScript to JavaScript:
```bash
npm run build
```

Or watch for changes during development:
```bash
npm run watch
```

### 3. Load in Figma
1. Open the Figma Desktop App.
2. Go to **Plugins** > **Development** > **Import plugin from manifest...**.
3. Select the `manifest.json` file in this directory.
# background-remover-figma-plugin
